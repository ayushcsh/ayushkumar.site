# Ayush Kumar

### Personal Portfolio Website

A modern, responsive portfolio showcasing my projects, technical skills, certifications, achievements, and journey as a software developer.

[Live Website](https://ayushkumar.site) - [Portfolio Studio](https://ayushkumar.site/studio)

</div>

---

## Tech Stack

* **Next.js** - React framework
* **TypeScript** - Type safety
* **Tailwind CSS** - Styling and UI
* **Sanity.io** - Headless CMS
* **Vercel** - Hosting and deployment
* **Next Themes** - Dark/light theme support
* **React Icons** - Icon library
* **Framer Motion** - Animations

---

## Features

* Responsive portfolio UI
* Dynamic project showcase
* Blog system powered by Sanity CMS
* Certifications and LeetCode badges section
* GitHub contribution graph
* Portfolio chatbot powered by AI
* Favorite songs and photos gallery
* Chess section and interactive components
* Dark and light themes
* SEO optimized pages
* Analytics integration

## Run Locally

```bash
git clone https://github.com/ayushcsh/ayushkumar.site.git
cd ayushkumar.site
npm install
npm run dev
```

Visit:

```text
http://localhost:3000
```

---

## Environment Variables

Create `.env.local` from `.env.example`, then add the required Sanity, GitHub, contact, and AI service keys for local development.

---

## Important Directories

| Folder/File            | Description                      |
| ---------------------- | -------------------------------- |
| `app/`                 | Next.js App Router pages         |
| `app/components`       | Reusable components              |
| `app/data`             | Static data and configuration    |
| `schemas/`             | Sanity content schemas           |
| `lib/sanity.client.ts` | Sanity configuration             |
| `lib/sanity.query.ts`  | GROQ queries                     |
| `public/`              | Images, music, and static assets |

---

## Build

```bash
npm run build
```
